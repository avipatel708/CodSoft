# 🌸 Iris Flower Classification Using Machine Learning

## 📖 Project Overview

This project focuses on classifying Iris flowers into different species using Machine Learning techniques. By analyzing flower measurements such as sepal length, sepal width, petal length, and petal width, the model accurately predicts the species of an Iris flower.

The project demonstrates a complete Machine Learning workflow including data preprocessing, exploratory data analysis (EDA), visualization, model training, evaluation, and prediction.

---

## 🎯 Objectives

- Analyze the Iris flower dataset
- Understand relationships between flower features
- Perform data cleaning and preprocessing
- Conduct Exploratory Data Analysis (EDA)
- Build multiple classification models
- Compare model performances
- Predict flower species accurately
- Extract meaningful insights from data

---

## 📊 Dataset

The dataset contains measurements of Iris flowers belonging to three species.

### Features Include:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width
- Species

### Target Classes

- Iris Setosa
- Iris Versicolor
- Iris Virginica

### Dataset Source

Iris Flower Dataset (Kaggle)
